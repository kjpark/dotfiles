# dotfiles

terminal config files, meet 2021.

no crazy magic anymore, just a simple reference repo.

## installation

in `~/.zshrc`
```
source ~/path/to/dotfiles/.zshrc
```

`.vimrc` from `dotfiles` dir:
```
ln -s "$(pwd)/.vimrc" ~/.vimrc
```
